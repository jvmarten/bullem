# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

We are committed to providing a welcoming and inclusive experience for everyone. We expect all participants to treat each other with respect and professionalism.

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing the project maintainer. All reports will be reviewed and handled confidentially.

## Enforcement

Project maintainers are responsible for enforcing this code of conduct. Participants who do not follow it may be removed from the project at the maintainers' discretion.
